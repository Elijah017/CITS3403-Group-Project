What is DevBoard?


DevBoard is a website tool designed to help teams visualise the progress of projects. Users can create project planners called boards and the owner of a DevBoard can add other users to it as helpers. Helpers are able to comment and assist in the progress of the project. Tasks can be split into different categories. For example categories can be: to do, started, under test, completed. In general, DevBoards is a great way to get some ideas if you are having trouble getting started.
